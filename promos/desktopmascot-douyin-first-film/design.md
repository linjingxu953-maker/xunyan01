# DesktopMascot Douyin Promo Design

## Purpose

Create the first short-form promo for DesktopMascot: a desktop AI assistant built over eight days with Codex, MiMo Code, and Claude Code. The video should feel like a candid build log that suddenly becomes a product teaser.

## Format

- Platform: Douyin
- Canvas: 1080 x 1920
- Duration target: 28 seconds
- Delivery: HyperFrames MP4, with a HeyGen-ready Chinese narration script

## Visual Identity

- Mood: late-night builder energy, desktop cockpit, animated assistant characters, practical workflow automation
- Background: inky warm black `#100F14`
- Panel: deep graphite `#191820`
- Raised panel: smoky violet-gray `#25232D`
- Primary text: warm paper `#F7EBDD`
- Secondary text: muted sand `#BCAFA0`
- Warm accent: amber `#FFB86B`
- Alert accent: red `#D94848`
- System accent: blue `#5FA8FF`
- Success accent: green `#73D39A`

## Typography

- Chinese headline: renderer-safe `sans-serif`
- Interface and code: renderer-safe `monospace`
- Use large, compressed short phrases for Douyin readability.
- Avoid small labels except for decorative metadata and code texture.

## Motion

- Rhythm: hook-PUNCH-build-PEAK-breathe-CTA
- Primary transition: fast blur/zoom through between related scenes
- Accent transition: overexposure flash for the hero reveal
- Each scene needs visible entrance animation and at least one ambient motion layer.

## Do Not

- Do not make it a generic SaaS launch video.
- Do not hide the characters behind tiny UI thumbnails.
- Do not use a blue-purple neon palette as the dominant theme.
- Do not over-explain features with paragraph text.
