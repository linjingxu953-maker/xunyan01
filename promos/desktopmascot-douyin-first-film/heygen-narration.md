# HeyGen Narration Script

## Voice Direction

Use a fast, conversational Chinese voice. The first sentence should sound like a slightly outrageous build-log hook, then settle into confident product-demo narration. Keep pauses after the hook and before the CTA.

## Script

我花了八天的时间，让我的朋友们在我的桌面上给我当牛马。

这八天里，我用 Codex、MiMo Code 和 Claude Code，开发了一个可以交互、还能接入工作流的桌面智能助手。

它能看懂屏幕，理解我圈出来的问题，帮我处理文件、查资料、写代码、做设计，甚至规划短视频流程。

最关键的是，它不是一个只会聊天的窗口。

它有角色，有记忆，有权限确认，也能把任务拆成一步一步执行。

这只是第一个宣传片。

接下来，我会把它做成真正住在桌面上的个人智能助理。

## Caption Beats

- 0.0s: 我花了8天
- 1.2s: 让朋友们在桌面上给我当牛马
- 4.0s: Codex + MiMo Code + Claude Code
- 7.0s: 一个可交互的桌面智能助手
- 11.0s: 看屏幕 / 懂意图 / 接工作流
- 16.0s: 文件、资料、代码、设计、短视频
- 22.0s: 不是聊天窗口，是桌面上的工作伙伴
- 25.0s: DesktopMascot
