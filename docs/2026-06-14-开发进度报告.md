## 三、Computer Use 架构

### 分工

| 角色 | 职责 |
|------|------|
| MiMo（Agent/Core） | 截图、窗口识别、鼠标键盘动作、浏览器/桌面操作、动作规划、工具执行管道 |
| Codex（UI） | 状态展示、权限确认、用户接管/中断、操作轨迹、结果反馈 |

### 事件接口（给 Codex）

```
ComputerUseStarted    → 任务开始
ScreenObserved        → 截图完成
ActionPlanned         → 动作规划完成
ActionExecuting       → 正在执行
ActionCompleted       → 执行完成
WaitingUserApproval   → 等待确认
UserTakeoverRequested → 用户接管
ComputerUseCompleted  → 任务完成
ComputerUseFailed     → 任务失败
```

### UI 控制面板壳子（Codex 下一步）

- 当前屏幕目标
- 计划动作列表
- 正在执行的动作
- 暂停/接管/终止按钮
- 权限确认入口
- 操作日志
