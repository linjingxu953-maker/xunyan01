## 完成内容

### 1. 工作流存储

#### IWorkflowStore 接口
- SaveAsync() - 保存工作流
- GetByIdAsync() - 按ID获取
- GetAllAsync() - 获取所有
- GetByStatusAsync() - 按状态获取
- DeleteAsync() - 删除
- CleanupAsync() - 清理旧数据
- SaveDefinitionAsync() - 保存定义
- GetDefinitionAsync() - 获取定义

#### FileWorkflowStore 实现
- JSON 文件存储
- 线程安全操作
- 自动清理

### 2. 持久化工作流引擎

#### PersistentWorkflowEngine
- 继承 IWorkflowEngine 接口
- 每步执行后自动保存
- 检查点恢复
- 状态持久化

#### 检查点恢复
- ResumeFromCheckpointAsync() - 从检查点恢复
- 自动找到未完成步骤继续执行

## 使用示例

```csharp
// 创建持久化引擎
var store = new FileWorkflowStore();
var engine = new PersistentWorkflowEngine(toolRegistry, store);

// 执行工作流（自动保存每步状态）
var workflow = engine.CreateWorkflow(definition);
await engine.ExecuteAsync(workflow);

// 从检查点恢复（断点续传）
var resumed = await engine.ResumeFromCheckpointAsync(workflow.Id);
```
