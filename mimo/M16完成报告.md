## 使用示例

```csharp
// 使用构建器创建工作流
var workflow = new WorkflowBuilder("我的任务")
    .AddStep("获取时间", "get_current_time")
    .AddStep("计算", "calculator", """{"expression": "2 + 2"}""")
    .Build();

// 使用引擎执行
var engine = new WorkflowEngine(toolRegistry);
var instance = engine.CreateWorkflow(workflow);
var result = await engine.ExecuteAsync(instance);

// 监听事件
engine.WorkflowEventOccurred += e => 
    Console.WriteLine($"[{e.EventType}] {e.Message}");
```
