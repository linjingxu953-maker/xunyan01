## 使用示例

```csharp
// 创建调度器
var scheduler = new AppTaskScheduler(toolRegistry);
await scheduler.StartAsync();

// 添加定时任务
var task = new ScheduledTask
{
    Name = "每小时执行",
    ToolName = "get_weather",
    Arguments = """{"city": "北京"}""",
    Type = ScheduleType.Interval,
    Interval = TimeSpan.FromHours(1)
};
await scheduler.AddTaskAsync(task);

// 立即执行
var execution = await scheduler.RunNowAsync(task.Id);

// 暂停/恢复
await scheduler.PauseTaskAsync(task.Id);
await scheduler.ResumeTaskAsync(task.Id);
```
