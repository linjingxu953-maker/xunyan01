# M30 完成报告：记忆确认接口

完成日期：2026-06-12

## 1. 交付内容

### 1.1 MemoryDecision 枚举

```csharp
public enum MemoryDecision
{
    /// <summary>保存到长期记忆</summary>
    Save,
    /// <summary>仅本次有效</summary>
    TempOnly,
    /// <summary>不保存</summary>
    Reject
}
```

### 1.2 MemoryConfirmationRequest 模型

```csharp
public record MemoryConfirmationRequest
{
    public string RequestId { get; init; }
    public string TaskId { get; init; }
    public MemoryType MemoryType { get; init; }
    public string Content { get; init; }
    public string Source { get; init; }
    public string Reason { get; init; }
    public Dictionary<string, string> Metadata { get; init; }
    public DateTime RequestedAt { get; init; }
}
```

### 1.3 MemoryConfirmationResponse 模型

```csharp
public record MemoryConfirmationResponse
{
    public string RequestId { get; init; }
    public MemoryDecision Decision { get; init; }
    public string? EditedContent { get; init; }
    public DateTime RespondedAt { get; init; }
}
```

### 1.4 IMemoryConfirmationPrompt 接口

```csharp
public interface IMemoryConfirmationPrompt
{
    Task<MemoryConfirmationResponse> PromptAsync(
        MemoryConfirmationRequest request,
        CancellationToken ct = default);
}
```

### 1.5 DefaultMemoryConfirmationPrompt 实现

功能：
1. 默认拒绝所有记忆保存请求
2. 可配置自动批准的记忆类型
3. 可配置自动拒绝的记忆类型
4. 用于 UI 未实现时的降级处理

### 1.6 MemoryConfirmationService 服务

功能：
1. 桥接 IMemoryStore 和 IMemoryConfirmationPrompt
2. 发布记忆保存请求事件到 ITaskEventStream
3. 根据用户决策保存/临时/拒绝记忆
4. 支持用户编辑记忆内容后保存
