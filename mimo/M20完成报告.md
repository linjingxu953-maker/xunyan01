## 完成内容

### 1. 缓存模型

#### CacheModels
- `CacheExpiration` - 过期策略（None/Absolute/Sliding/FileDependency）
- `CacheEntry<T>` - 缓存条目
- `CacheStatistics` - 缓存统计
- `CacheOptions` - 缓存选项

### 2. 缓存实现

#### ICache 接口
- Get<T>() - 获取缓存
- Set<T>() - 设置缓存
- Remove() - 移除缓存
- Clear() - 清空缓存
- Contains() - 检查存在
- GetStatistics() - 获取统计
- Cleanup() - 清理过期
- RemoveByTag() - 按标签移除

#### MemoryCache 实现
- 线程安全操作
- 绝对/滑动过期
- 容量限制（自动驱逐）
- 命中率统计

### 3. 缓存装饰器

#### CachedService<T>
- GetOrCreate() - 获取或创建
- GetOrCreateAsync() - 异步获取或创建
- Invalidate() - 使缓存失效
- InvalidateAll() - 使所有缓存失效

#### CacheExtensions
- WithCache() - 带缓存执行
- WithCacheAsync() - 异步带缓存执行

## 使用示例

```csharp
// 基本用法
var cache = new MemoryCache();
cache.Set("key1", "value1", CacheExpiration.Absolute, TimeSpan.FromMinutes(5));
var value = cache.Get<string>("key1");

// 带装饰器
var cachedService = new CachedService<IMyService>(myService, cache);
var result = await cachedService.GetOrCreateAsync("data", async () => await FetchDataAsync());

// 扩展方法
var result = await cache.WithCacheAsync("key", async () => await ExpensiveOperation());
```
