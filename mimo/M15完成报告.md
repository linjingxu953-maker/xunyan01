## 使用示例

```csharp
// 注册工具
var registry = new ToolRegistry();
registry.Register(new GetCurrentTimeTool());
registry.Register(new CalculatorTool());

// 执行工具
var result = await registry.ExecuteAsync(new ToolCallRequest
{
    ToolName = "calculator",
    Arguments = """{"expression": "2 + 3 * 4"}"""
});

// 搜索工具
var tools = registry.Search("math");

// 获取调用日志
var logs = await registry.GetCallLogsAsync();
```
