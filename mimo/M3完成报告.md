## 完成内容

### 1. 系统托盘

- `TrayIcon` 集成，含"显示/隐藏"、"唤起输入"、"退出"菜单项
- 托盘图标由 `TrayIconFactory.CreateDefaultIcon()` 生成
- 点击托盘图标可切换窗口显隐
- 工具提示显示快捷键信息

### 2. 全局快捷键

- `IGlobalHotkeyService` 接口定义
- `WindowsGlobalHotkeyService` Windows 实现
- 默认快捷键注册
- 快捷键触发时自动唤起窗口并展开聊天面板

### 3. 窗口位置记忆

- `IWindowPlacementStore` 接口定义
- `JsonWindowPlacementStore` 实现 —— JSON 文件持久化
- 窗口关闭/退出时自动保存位置和尺寸
- 启动时恢复上次位置（带合理性校验）
- 拖动/缩放后防抖保存（700ms 延迟）

### 4. 窗口交互

- 关闭按钮 → 隐藏到托盘（不退出）
- 托盘退出 → 完全退出
- 窗口状态管理（显示/隐藏/最小化恢复）

## 涉及文件

| 文件 | 说明 |
|------|------|
| `Services/DesktopShellService.cs` | 桌面外壳服务（托盘、快捷键、位置记忆集成） |
| `Services/WindowsGlobalHotkeyService.cs` | Windows 全局快捷键 |
| `Services/IGlobalHotkeyService.cs` | 快捷键接口 |
| `Services/JsonWindowPlacementStore.cs` | 窗口位置 JSON 存储 |
| `Services/IWindowPlacementStore.cs` | 窗口位置存储接口 |
| `Services/WindowPlacementState.cs` | 窗口位置数据模型 |
| `Services/TrayIconFactory.cs` | 托盘图标工厂 |

## 验证方式

```powershell
dotnet run --project src/DesktopMascot.App
```

1. 悬浮窗口正常显示在桌面
2. Ctrl+Shift+M（默认）唤起聊天面板
3. 系统托盘出现图标，右键菜单可用
4. 关闭窗口 → 隐藏到托盘，任务栏不显示
5. 托盘退出 → 完全退出
6. 重启后窗口回到上次位置
