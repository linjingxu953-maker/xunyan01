## 完成内容

### 1. 插件接口

#### IPlugin 接口
- `Metadata` - 插件元数据
- `State` - 插件状态
- `InitializeAsync()` - 初始化
- `EnableAsync()` - 启用
- `DisableAsync()` - 禁用
- `UnloadAsync()` - 卸载

#### PluginBase 基类
- 提供默认实现
- 配置加载/保存
- 数据目录管理

### 2. 插件管理

#### PluginLoader
- 扫描插件目录
- 从程序集加载插件
- ZIP 包安装
- 插件卸载

#### PluginRegistry
- 插件注册/注销
- 插件启用/禁用
- 事件通知
- 工具提供者管理

### 3. 插件工具

#### IPluginTool 接口
- 插件专用工具接口
- 与 Agent 工具系统解耦

#### IToolProvider 接口
- 工具提供者接口
- 插件可导出工具

### 4. 示例插件

| 插件 | 功能 |
|------|------|
| QuotesPlugin | 随机编程名言 |
| WeatherPlugin | 天气查询（模拟） |

## 开发自定义插件

```csharp
public class MyPlugin : PluginBase, IToolProvider
{
    public override PluginMetadata Metadata => new()
    {
        Id = "my.plugin",
        Name = "我的插件",
        Version = "1.0.0"
    };

    public IEnumerable<object> GetTools()
    {
        return new object[] { new MyTool() };
    }
}

public class MyTool : IPluginTool
{
    public string Name => "my_tool";
    public string Description => "我的工具";
    public string ParametersSchema => "{}";

    public Task<PluginToolResult> ExecuteAsync(string arguments, CancellationToken ct)
    {
        return Task.FromResult(new PluginToolResult
        {
            Success = true,
            Content = "Hello from plugin!"
        });
    }
}
```
