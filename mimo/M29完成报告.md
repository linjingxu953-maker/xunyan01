## 1. 交付内容

### 1.1 PromptPermissionType 枚举

定义了 12 种权限类型：

```csharp
public enum PromptPermissionType
{
    // 文件操作
    FileRead,
    FileWrite,
    FileDelete,

    // 命令执行
    CommandExecute,

    // 上下文读取
    ScreenCapture,
    BrowserRead,
    WindowRead,
    SelectedTextRead,

    // 记忆操作
    MemorySave,
    MemoryDelete,

    // 外部调用
    ApiCall,
    WebhookSend
}
```

### 1.2 PromptRiskLevel 枚举

```csharp
public enum PromptRiskLevel
{
    Low,      // 低风险：读取文件、读取窗口标题
    Medium,   // 中风险：写入文件、执行命令
    High      // 高风险：删除文件、执行危险命令
}
```

### 1.3 PermissionPromptRequest 模型

```csharp
public record PermissionPromptRequest
{
    public string RequestId { get; init; }
    public string TaskId { get; init; }
    public PromptPermissionType PermissionType { get; init; }
    public string Scope { get; init; }
    public string Reason { get; init; }
    public string CommandOrPath { get; init; }
    public PromptRiskLevel RiskLevel { get; init; }
    public Dictionary<string, string> Metadata { get; init; }
    public DateTime RequestedAt { get; init; }
}
```

### 1.4 PermissionPromptResponse 模型

```csharp
public record PermissionPromptResponse
{
    public string RequestId { get; init; }
    public PermissionDecision Decision { get; init; }
    public string? DenyReason { get; init; }
    public DateTime RespondedAt { get; init; }
}
```

### 1.5 IPermissionPrompt 接口

```csharp
public interface IPermissionPrompt
{
    Task<PermissionPromptResponse> PromptAsync(
        PermissionPromptRequest request,
        CancellationToken ct = default);

    bool HasPermission(PromptPermissionType type, string scope);

    void RevokePermission(PromptPermissionType type, string scope);
}
```

### 1.6 DefaultPermissionPrompt 实现

功能：
1. 默认拒绝所有权限请求
2. 可配置自动批准的权限类型
3. 支持手动授予/撤销权限
4. 用于 UI 未实现时的降级处理

### 1.7 PermissionConfirmationService 服务

功能：
1. 桥接 IPermissionManager 和 IPermissionPrompt
2. 发布权限请求事件到 ITaskEventStream
3. 记录审计日志
4. 映射权限类型到 PermissionLevel
