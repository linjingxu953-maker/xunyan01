## 技术实现

### AgentOrchestrator ReAct 循环

```
1. 构建系统提示 + 用户输入
2. 调用 LLM（带工具定义）
3. 解析响应
   - 如果是最终响应 → 返回结果
   - 如果包含工具调用 → 执行工具 → 继续循环
4. 重复直到完成或达到最大迭代次数
```

### 工具调用格式

```json
{
  "tool_call": {
    "name": "tool_name",
    "arguments": {...}
  }
}
```
