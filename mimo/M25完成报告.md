# M25 完成报告：工具执行管道

完成日期：2026-06-12

## 1. 交付内容

### 1.1 ToolExecutionPipeline 服务

功能：
1. **工具存在性检查** - 验证工具是否已注册
2. **工具启用检查** - 验证工具是否启用
3. **权限检查** - 根据工具的 RequiredPermission 请求权限确认
4. **权限确认** - 调用 PermissionConfirmationService 请求用户确认
5. **工具执行** - 调用工具的 ExecuteAsync 方法
6. **异常处理** - 捕获工具执行异常
7. **事件发布** - 发布 ToolCallStarted/ToolCallCompleted 事件
8. **日志记录** - 工具执行结果自动记录

### 1.2 执行流程

```text
ToolExecutionPipeline.ExecuteAsync
  ├── 1. 检查工具是否存在
  ├── 2. 检查工具是否启用
  ├── 3. 检查是否需要权限
  │   ├── 发布 ToolCallStarted 事件
  │   ├── 调用 PermissionConfirmationService
  │   └── 检查权限决策
  ├── 4. 执行工具
  │   └── 异常处理
  ├── 5. 发布 ToolCallCompleted 事件
  └── 6. 返回结果
```

### 1.3 权限映射

| PermissionLevel | PromptPermissionType | PromptRiskLevel |
|-----------------|---------------------|-----------------|
| L0_Chat | 无需权限 | - |
| L1_WindowTitle | WindowRead | Low |
| L2_ScreenBrowser | ScreenCapture | Low |
| L3_FileRead | FileRead | Low |
| L4_FileWrite | FileWrite | Medium |
| L5_CommandExec | CommandExecute | High |
| L6_Forbidden | CommandExecute | High |

## 2. 使用示例

```csharp
// 注入 ToolExecutionPipeline
var pipeline = service.GetRequiredService<ToolExecutionPipeline>();

// 创建工具调用请求
var request = new ToolCallRequest
{
    ToolName = "read_file",
    Arguments = "{\"path\": \"test.txt\"}",
    TaskId = "task_001"
};

// 执行工具（自动处理权限检查）
var response = await pipeline.ExecuteAsync(request);
```
