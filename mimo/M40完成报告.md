## 完成内容

### 1. ComputerUseTool — 鼠标键盘控制工具

支持的操作：
- `click` / `double_click` / `right_click` — 鼠标点击
- `type` — 键盘输入
- `hotkey` — 快捷键组合
- `scroll` — 滚动
- `move_cursor` — 移动光标
- `drag` — 拖拽
- `screenshot` — 截图
- `get_cursor_pos` / `get_screen_size` — 获取光标位置/屏幕尺寸
- `maximize_window` / `minimize_window` / `restore_window` — 窗口操作
- `get_active_window` — 获取当前窗口信息

### 2. ComputerUseEvent — 事件模型

9 种事件类型（UI 层根据这些事件更新控制面板）：

| 事件 | 说明 |
|------|------|
| ComputerUseStarted | 任务开始 |
| ScreenObserved | 截图完成 |
| ActionPlanned | 动作规划完成 |
| ActionExecuting | 正在执行 |
| ActionCompleted | 执行完成 |
| WaitingUserApproval | 等待确认 |
| UserTakeoverRequested | 用户接管 |
| ComputerUseCompleted | 任务完成 |
| ComputerUseFailed | 任务失败 |

### 3. ComputerUseOrchestrator — 编排器

完整流程：
```
用户请求
  → 截图（ScreenObserved）
  → LLM 规划动作（ActionPlanned）
  → 逐个执行（ActionExecuting → ActionCompleted）
  → 敏感操作等待确认（WaitingUserApproval）
  → 任务完成（ComputerUseCompleted）
```

支持：
- 暂停/恢复
- 用户接管
- 动作规划（LLM + 视觉）
- 错误处理
