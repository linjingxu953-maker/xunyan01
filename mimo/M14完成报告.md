## 完成内容

### 1. 错误模型

#### ErrorModels
- `ErrorType` - 错误类型（Network/Api/FileOperation/Permission等）
- `AppException` - 应用异常
- `OperationResult` - 操作结果
- `OperationResult<T>` - 带返回值的操作结果

### 2. 错误处理器

#### ErrorHandler
- 全局错误处理器
- 异常分类（ClassifyError）
- 错误事件通知

#### ErrorContext
- 错误上下文信息
- 异常、来源、任务ID、时间戳

### 3. 安全执行器

#### SafeExecutor
- `ExecuteAsync<T>()` - 异步执行（带返回值）
- `ExecuteAsync()` - 异步执行（无返回值）
- `Execute()` - 同步执行

#### RetryPolicy 重试策略
- MaxRetries - 最大重试次数
- InitialDelay - 初始延迟
- MaxDelay - 最大延迟
- BackoffMultiplier - 退避倍数
- ShouldRetry - 自定义重试条件

## 使用示例

```csharp
// 带重试的执行
var result = await SafeExecutor.ExecuteAsync(
    async ct => await CallExternalApi(),
    new RetryPolicy { MaxRetries = 3 },
    errorHandler);

if (!result.Success)
{
    Logger.Error(result.ErrorMessage);
}

// 操作结果模式
var result = await SafeExecutor.ExecuteAsync<string>(async ct =>
{
    var data = await ReadFile(path);
    return data;
});

return result.Match(
    onSuccess: data => ProcessData(data),
    onFailure: error => HandleError(error));
```
