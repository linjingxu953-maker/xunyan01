## 完成内容

### 1. 上下文提供者

#### IContextProvider 接口
- `GetActiveWindowContextAsync()` - 获取当前窗口信息
- `GetSelectedTextAsync()` - 获取选中文本
- `ReadFileAsync()` - 读取文件内容
- `GetFullContextAsync()` - 获取完整上下文

#### WindowsContextProvider 实现
- Windows API 调用获取窗口标题
- 进程名获取
- 文件读取（带大小限制）
- 跨平台兼容

#### MockContextProvider
- 测试用模拟实现
- 可配置返回值

### 2. 上下文工具

| 工具 | 功能 |
|------|------|
| get_active_window | 获取当前窗口标题和应用名 |
| read_file | 读取指定路径文件内容 |

### 3. ContextSnapshot 模型

```csharp
public class ContextSnapshot
{
    string ActiveWindowTitle;    // 窗口标题
    string ActiveApplication;    // 应用名
    string SelectedText;         // 选中文本
    string? BrowserUrl;         // 浏览器 URL
    string? BrowserTitle;       // 浏览器标题
    string? FileContent;        // 文件内容
    string? FilePath;           // 文件路径
    DateTime CapturedAt;        // 采集时间
}
```
